# KNOWLEDGE GRAPHS AND HYBRID INTELLIGENCE IN CLINICAL SETTING
Extension of the HI ontology and proposition of a novel neuro-symbolic method for identifying latent failure points in clinical HI systems for the Knowledge Graphs and Sematic Technologies Course at VU Amsterdam. 

The project was made by: Joram van Beem and Lea Strazisar. 

## Repository Contents
- `Modolled_Individuals_Assigment.ttl` — extended HI ontology (TBox + ABox)
- `HI_Ontology_KG_Consumption.ipynb` — Colab notebook with basic measure SPARQL queries, 
   graph metrics and link prediction
- `HI_visualize.py` - script for visualizing the ontology and provinding the figure in the plot

## Ontology Statistics
- 107 classes
- 82 object properties
- 339 individuals
- 2229 triples
- 4388 inferences

## How to Run
1. Open `HI_Ontology_KG_Consumption.ipynb` in Google Colab
2. Upload the file if needed
3. Run all cells
